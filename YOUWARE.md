# 新年烟花

## 项目概览
一款国潮风格的新年烟花网页互动游戏，用户点击屏幕即可发射多种形状的烟花，烟花绽放时拼出个性化祝福语，并支持生成静态新年贺卡图片。

## 技术栈
- **框架**: React 18 + TypeScript + Vite 7
- **样式**: Tailwind CSS 3.4
- **动画**: Framer Motion + HTML5 Canvas
- **音频**: Web Audio API (合成音效)

## 核心功能
1. **烟花互动** - 点击屏幕发射烟花，支持圆形、爱心、星形、环形、菊花、文字等多种形状
2. **祝福语绽放** - 每隔几次点击，烟花会在空中拼出随机祝福语（新年快乐、恭喜发财等）
3. **音效系统** - Web Audio API 合成背景音乐（五声音阶中国风旋律）、爆炸声、发射声、爆竹声
4. **水墨转场** - 每15次烟花爆炸自动触发水墨风格动画转场，也可手动触发
5. **贺卡生成** - 选择祝福语、输入署名，一键生成并下载精美国潮风新年贺卡

## 设计风格
- 复古国潮配色（中国红 + 金色 + 朱砂红）
- 红金交织丝绸纹理背景
- 水墨风格动画转场
- 金色边框角花装饰

## 项目架构
```
src/
├── App.tsx                    # 主页面，整合所有组件
├── components/
│   ├── FireworksCanvas.tsx    # 烟花画布主组件
│   ├── SilkBackground.tsx     # 丝绸纹理背景
│   ├── InkTransition.tsx      # 水墨转场动画
│   └── CardGenerator.tsx      # 贺卡生成器
├── utils/
│   ├── fireworks.ts           # 烟花粒子引擎
│   └── audio.ts               # 音频合成管理
├── index.css                  # 全局样式
└── main.tsx                   # 应用入口
```
